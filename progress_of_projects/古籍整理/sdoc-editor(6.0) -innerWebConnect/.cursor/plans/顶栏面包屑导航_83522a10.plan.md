# 顶栏面包屑（project-join-request / project-members）

## 目标

- 用 **面包屑** 替代当前单列 `nav-links`（如「项目」或「返回项目」）。
- 层级之间使用 **`>`** 分隔；可点击层级使用现有 `nav-link` 风格，**当前页**为纯文本（不嵌套链接）。

## 面包屑层级（建议）

| 页面 | 面包屑 |
|------|--------|
| [project-members.html](C:\Users\25283\Desktop\古籍整理\sdoc-editor(6.0) -innerWebConnect\project-members.html) | `首页` > `项目` > `成员管理` |
| [project-join-request.html](C:\Users\25283\Desktop\古籍整理\sdoc-editor(6.0) -innerWebConnect\project-join-request.html) | `首页` > `项目` > `申请加入` |

- **首页** → `index.html`
- **项目** → `project.html`
- **末级**（成员管理 / 申请加入）→ `<span class="nav-breadcrumb-current">`，无 `href`

## HTML 结构示例

在 `top-nav` 内保留 `nav-logo`，将原：

```html
<div class="nav-links">...</div>
```

替换为带语义与可访问性的导航，例如：

```html
<nav class="nav-links nav-breadcrumb" aria-label="面包屑">
  <a href="index.html" class="nav-link">首页</a>
  <span class="nav-breadcrumb-sep" aria-hidden="true">&gt;</span>
  <a href="project.html" class="nav-link">项目</a>
  <span class="nav-breadcrumb-sep" aria-hidden="true">&gt;</span>
  <span class="nav-breadcrumb-current">成员管理</span>
</nav>
```

`project-join-request.html` 末级文案为「申请加入」。

## 样式

在 [css/surface-pages.css](C:\Users\25283\Desktop\古籍整理\sdoc-editor(6.0) -innerWebConnect\css\surface-pages.css)（或 [css/common.css](C:\Users\25283\Desktop\古籍整理\sdoc-editor(6.0) -innerWebConnect\css\common.css)）中新增：

- `.nav-breadcrumb`：`display:flex; align-items:center; flex-wrap:wrap; gap` 略小于原 `nav-links`，保证 `>` 与文字对齐。
- `.nav-breadcrumb-sep`：浅色、略小字号，与顶栏白字对比协调（如 `rgba(255,255,255,0.45)`）。
- `.nav-breadcrumb-current`：与链接区分（如 `opacity:0.95` 或略浅），**无 hover 背景**（非链接）。

避免破坏其他仍使用 `.nav-links` 的页面：新类名加在 **额外 class** 上（`nav-links nav-breadcrumb`），仅这两页使用面包屑结构。
