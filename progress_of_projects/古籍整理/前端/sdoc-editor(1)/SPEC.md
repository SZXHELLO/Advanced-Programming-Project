# 古籍在线编辑器 (SDOC Editor) - 项目规范

## 1. 概念与愿景

这是一款专为古籍整理工作设计的专业在线编辑器，融合传统书房的雅致与现代数字工具的便捷。界面设计灵感来源于古籍的书页质感与毛笔书写的韵味，同时保持功能的专业性与高效性。用户能够对古籍扫描页面进行精细的文本标注、属性编辑、造字处理，最终输出符合SDOC标准的XML文件。

## 2. 设计语言

### 美学方向
- **主题**: 新中式书房风格 - 将传统书房的沉静雅致与现代界面设计融合
- **质感**: 宣纸纹理背景、水墨晕染效果、印章风格的图标
- **氛围**: 宁静、专业、典雅

## 4. 功能与交互

### 4.1 项目管理
- **新建项目**: 创建空白项目，填写元数据（标题、作者、书籍信息）
- **打开项目**: 加载本地XML文件或从服务器获取
- **保存项目**: 自动保存 + 手动保存，支持导出XML
- **导入图片**: 支持批量导入古籍扫描图片

### 4.2 古籍图片展示
- **多页管理**: 支持多张页面，按顺序浏览
- **缩放控制**: 25% - 400% 缩放，滚轮缩放 + 按钮控制
- **平移导航**: 拖拽平移，保持在视口内
- **页码导航**: 首页/上一页/下一页/末页 + 直接跳转

### 4.3 文本标注功能
**标注类型**:
| 类型 | 描述 | 视觉样式 |
|------|------|---------|
| 文字框选 | 框选单个或多个字符 | 半透明矩形边框 |
| 下划线 | 为文字添加下划线 | 直线/波浪线 |
| 高亮 | 为文字添加背景色 | 半透明背景色 |
| 删除线 | 标记需要删除的内容 | 删除线样式 |
| 插入标记 | 标记插入位置 | 插入符号 |

**标注操作**:
1. 选择标注工具
2. 在图片上拖拽创建标注区域
3. 自动弹出右侧属性面板
4. 输入简体字、注释等信息
5. 保存标注

### 4.4 属性编辑面板
**字段**:
- **简体字**: 输入框，用于输入对应的简体字
- **标注颜色**: 颜色选择器
- **标注样式**: 下拉选择（框选/下划线/高亮/删除线）
- **注释**: 多行文本框
- **原文**: 显示原始古字（只读）

### 4.5 造字功能
**造字流程**:
1. 点击"造字"按钮打开造字面板
2. 在画布上绘制新的汉字字形
3. 自动分配Unicode编码
4. 预览生成效果
5. 保存到本地字库/项目

**造字面板元素**:
- 绘制画布 (256x256px)
- 画笔工具 (粗细可调)
- 橡皮擦工具
- 清除按钮
- 预览区域
- Unicode编码显示
- 保存/取消按钮

### 4.6 XML导出
**导出内容**:
- 完整的SDOC格式XML
- 图片引用（相对路径）
- 所有标注信息
- 元数据

**导出格式**:
```xml
<?xml version="1.0" encoding="UTF-8"?>
<article id="..." type="1" version="1.0">
  <head>
    <title name="...">...</title>
    <authors>
      <author name="..." type="0"/>
    </authors>
    <book name="..." pages="..."/>
    <date year="..."/>
  </head>
  <content page_mode="1">
    <page id="p1" page_no="1">
      <panel>
        <textfield>...</textfield>
      </panel>
    </page>
  </content>
  <view count="...">
    <svg id="v1" page_no="1" width="..." height="...">
      <text x="..." y="..." fill="...">...</text>
    </svg>
  </view>
  <sources>
    <source src="..." pageno="1"/>
  </sources>
</article>
```

## 5. 组件清单

### 5.1 工具栏按钮
- **默认态**: 图标 + 文字，背景透明
- **悬停态**: 背景色变浅，轻微上浮阴影
- **激活态**: 背景色为主色调，图标高亮
- **禁用态**: 灰色，无交互

### 5.2 输入框
- **默认态**: 白色背景，淡边框
- **聚焦态**: 边框变为主色调，添加阴影
- **错误态**: 边框变红，显示错误提示
- **禁用态**: 灰色背景

### 5.3 颜色选择器
- 预设颜色网格
- 当前选中颜色高亮
- 支持自定义颜色输入

### 5.4 标注卡片
- 显示标注预览
- 原文 + 简体字对照
- 快速操作按钮（编辑/删除）

### 5.5 模态框
- 半透明遮罩背景
- 居中卡片式弹窗
- 标题 + 内容 + 操作按钮
- ESC键关闭

### 5.6 页面缩略图
- 当前页高亮边框
- 可拖拽排序
- 右键菜单（删除/复制）

## 6. 技术方案

### 前端技术栈
- **HTML5**: 语义化结构
- **CSS3**: 自定义属性变量、Flexbox、Grid
- **JavaScript**: 原生ES6+，模块化组织
- **Canvas API**: 造字绘制、图片标注
- **SVG**: 标注矢量显示

### 数据结构
```javascript
// 项目结构
Project {
  id: string,
  metadata: {
    title: string,
    authors: Author[],
    book: Book,
    date: Date
  },
  pages: Page[],
  annotations: Annotation[],
  customChars: CustomChar[]
}

// 页面结构
Page {
  id: string,
  pageNo: number,
  imageSrc: string,
  width: number,
  height: number
}

// 标注结构
Annotation {
  id: string,
  pageId: string,
  type: 'box' | 'underline' | 'highlight' | 'strikethrough',
  rect: { x, y, width, height },
  color: string,
  simplifiedChar: string,
  note: string,
  originalChars: string[]
}

// 造字结构
CustomChar {
  id: string,
  unicode: number,
  name: string,
  imageData: string
}
```

### 文件格式
- **项目文件**: JSON格式 (.sdocproj)
- **导出格式**: XML格式 (SDOC标准)
- **图片格式**: 支持 PNG/JPG/WebP

### 本地存储
- IndexedDB: 项目数据、标注数据
- LocalStorage: 用户偏好设置
